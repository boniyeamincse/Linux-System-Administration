# Lab 02: Virtual Environment Setup

## Scenario
You need to set up a safe environment to practice big changes without breaking your main computer.

## Steps
1.  **Download Hypervisor**: Go to the VMware or VirtualBox website and download the installer.
2.  **Install Hypervisor**: Run the installer and verify it opens.
3.  **Download ISO**:
    - Ubuntu: [ubuntu.com/download/server](https://ubuntu.com/download/server)
    - RHEL/Rocky: [rockylinux.org/download](https://rockylinux.org/download)
4.  **Create New VM**:
    - Select "New Virtual Machine".
    - Choose "Linux" > "Ubuntu 64-bit" or "Red Hat 9".
    - Allocate 2 CPU cores, 4GB RAM, 25GB Storage.
    - Attach the ISO file to the virtual CD/DVD drive.

## Verification
- You should see the VM in your inventory list.
- Powering on the VM should boot into the Installation menu.
