# Lab 16: Firewalls

## Objective
Secure the network by filtering traffic.

## Tasks
1.  Check the status of `firewalld`.
2.  List the currently allowed services.
3.  Add the `http` service to the firewall persistently.
4.  Add port 8080/tcp manually.
5.  Reload the firewall.
6.  Check the current SELinux mode.
