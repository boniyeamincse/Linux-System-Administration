# Lab 04: File System Management

## Objective
Become comfortable with Creating, Moving, Copying, and Deleting files and directories.

## Tasks
1.  Open Terminal.
2.  Check your current directory.
3.  Create a directory named `Training_Lab`.
4.  Navigate into `Training_Lab`.
5.  Create three empty files: `file1.txt`, `file2.txt`, `file3.txt`.
6.  List the files to verify.
7.  Create a subdirectory named `Backup`.
8.  Copy `file1.txt` into `Backup`.
9.  Rename `file2.txt` to `notes.txt`.
10. Delete `file3.txt`.
11. Return to your home directory and remove the entire `Training_Lab` directory (Be Careful!).
