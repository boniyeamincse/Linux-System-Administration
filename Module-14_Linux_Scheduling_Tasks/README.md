# Module 14: Linux Scheduling Tasks

## Topics Covered
- Cron and Crontab
- Cron Job Syntax: `min hour day month week cmd`
- User vs System Cron (`/var/spool/cron` vs `/etc/crontab`)
- `at` command for one-time tasks

## Syntax
`* * * * * command`
- Minute (0-59)
- Hour (0-23)
- Day of Month (1-31)
- Month (1-12)
- Day of Week (0-7)
