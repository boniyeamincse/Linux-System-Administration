# Linux System Administration (Ubuntu & RHEL)

**Course Duration:** 40 Hours | **Total Modules:** 16 | **Training Method:** Offline/Online

## Course Overview
Linux Systems Administrators configure, implement, and deploy systems, manage security, backups, and users. This course prepares students to become competent Linux System Administrators with analytical and problem-solving skills throughout 16 intensive modules covering everything from installation to advanced security and automation.

## Course Objectives
By the end of this course, students will be able to:
- Understand UNIX/Linux architecture, Kernel, Shell, and distributions.
- Install Linux (MBR/GPT) and configure post-installation settings.
- Utilize Desktop (GNOME) and Command Line (Terminal, Shell) environments.
- Manage files, directories, and navigate the Linux File System.
- Use text processing tools (grep, find, pipes, redirection, vi/vim).
- Administer Users, Groups, Sudo privileges, and Password Policies.
- Secure files with Permissions, Ownership, SUID/SGID, and ACLs.
- Manage Boot Process, Systemd services, GRUB2, and reset root passwords.
- Manage File Systems, Partitions (fdisk/gdisk), and Mounting (/etc/fstab).
- Configure Networking (IPv4, nmcli, nmtui, DNS, Hostname).
- Manage Packages using RPM, DNF, APT, and Local/Public Repositories.
- Configure OpenSSH for secure remote administration (Keys, Restrictions).
- Automate tasks using Cron and Crontab.
- Perform Backups, Archiving (tar, gzip, rsync), and analyze System Logs.
- Implement Security using Firewalld, Zones, and SELinux.
