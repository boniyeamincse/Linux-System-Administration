# Module 12: Linux Package Management System

## Topics Covered
- RPM vs DEB
- `rpm` command (Low level)
- `dnf` / `yum` (RHEL/CentOS High level)
- `apt` (Ubuntu/Debian High level)
- Repositories (Local and EPEL)

## Commands
### RHEL/CentOS
- `dnf install package`
- `dnf update`
- `dnf remove package`

### Ubuntu
- `apt update`
- `apt install package`
- `apt remove package`
