# Lab 01: System Identification

## Scenario
You are a new System Administrator. You have been given access to a server, but you don't know what it is running. Your task is to identify the Operating System, Kernel version, and Hardware architecture.

## Instructions
1.  Open your terminal.
2.  Run the command to find out the Kernel version.
3.  Run the command to identify the Linux Distribution name and version.
4.  Find out how much memory (RAM) is available on the system.
5.  Document your findings.

## Expected Output
- Kernel: `5.x.x` or similar.
- Distro: `Ubuntu 22.04` or `RHEL 9`.
- RAM: `8GB` (example).
