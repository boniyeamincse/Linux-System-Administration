# Module 06: Linux Text Editors

## Topics Covered
- Introduction to 'vi/vim' and 'nano'
- Working with Different 'vi/vim' Modes
  - **Command Mode**: Navigation, copy, paste, delete.
  - **Insert Mode**: Typing text.
  - **Last Line Mode**: Saving, quitting.
- Editing, Replacing, Searching with 'vi/vim'

## Vim Cheat Sheet
- `i`: Insert mode
- `Esc`: Exit Insert mode
- `:w`: Save
- `:q`: Quit
- `:wq`: Save and Quit
- `dd`: Delete line
- `u`: Undo
- `/pattern`: Search for pattern
