# Lab 06: Editing with Vim

## Objective
Master the basics of the Vi/Vim editor.

## Tasks
1.  Open a new file named `vim_practice.txt` using vim.
2.  Press `i` to enter Insert Mode.
3.  Type:
    ```
    Hello World.
    This is my first vim file.
    I love Linux.
    ```
4.  Press `Esc` to go back to Command Mode.
5.  Save the file (`:w`).
6.  Delete the second line (`dd`).
7.  Undo the deletion (`u`).
8.  Go to the end of the file (`G`) and add a new line: "The End".
9.  Save and Quit (`:wq`).
