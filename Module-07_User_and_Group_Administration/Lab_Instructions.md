# Lab 07: Managing Users and Groups

## Objective
Create users and groups, and assign users to groups.

## Tasks
1.  Create a new group named `sales`.
2.  Create a new user named `john` with a home directory.
3.  Set a password for `john`.
4.  Add `john` to the `sales` group.
5.  Create another user `sarah` and set her primary group to `sales` upon creation.
6.  Verify group memberships.
7.  Lock `john`'s account so he cannot login.
