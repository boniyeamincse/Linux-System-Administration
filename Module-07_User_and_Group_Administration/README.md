# Module 07: User and Group Administration

## Topics Covered
- Users and Groups Introduction
- `/etc/passwd`, `/etc/shadow`, `/etc/group`
- Primary Groups and Supplementary Groups
- Gaining Super user Access (`su`, `sudo`)
- Managing User Accounts (`useradd`, `usermod`, `userdel`)
- Managing Group Accounts (`groupadd`, `groupmod`)
- Password Policy

## Commands
- `useradd -m -s /bin/bash username`
- `passwd username`
- `usermod -aG groupname username` (Add to supplementary group)
- `id username` (Check info)
